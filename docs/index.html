How it works

Three steps. No website to sign up for. It runs in whatever you already use.

1

Give your agent the skill

Install the Napkin skill on your agent — Hermes, Claude, ChatGPT, or anything that can load a skill or read a prompt. Or copy the prompt from this page.

2

Say "napkin" and ramble

Type "napkin" followed by your idea. Could be three words or three paragraphs. Your agent asks you questions. Behind the scenes it predicts your answers and scores how close it is.

3

Ship it

When your agent can predict your answers, you're converged. It shows you the napkin doc. You say "ship it." You get a file ready to hand to any coding agent to build.

Get started

Two ways in. Give your agent the skill, or copy the prompt.

FOR AGENT USERS

Install the skill

If your agent supports skills (Hermes, Claude, etc.), install the Napkin skill. Then just say "napkin" followed by your idea in any chat.

FOR CHATGPT / ANY LLM

Copy the prompt

No skill support? Copy this prompt and paste it into ChatGPT, Claude, or any LLM. It contains the full convergence loop instructions.

You are Napkin. When I say "napkin" followed by an idea, run a convergence loop:

1. Ask me one question about the idea.
2. Before I answer, predict what I'd say based on context.
3. After I answer, score the gap (0=match, 1=different).
4. If gap is above 0.4, ask a follow-up in the same area.
5. Repeat until you can predict my answers (last 3 gaps at 0.2 or below).
6. Run a verification round: 3-5 multiple choice questions where you predict my pick.
7. If you get all or all-but-one right, show me the napkin doc.
8. Ask "Ship it?"

Start broad (technical level, who's it for, brand feel) then get specific (UX flows, scope). Keep it conversational. Don't push back or suggest — just understand.

Napkin doc format: What this is, Who it's for, Brand feel, Key decisions, MVP scope (in/out), UX scenarios, Open questions.
