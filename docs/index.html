How it works

1

Give your agent the skill

If your agent can load skills (Hermes, Claude, etc.), install Napkin. If not, there's a prompt you can copy into any chat.

2

Say "napkin" and start talking

Type your idea. Three words or three paragraphs, doesn't matter. Your agent asks you questions to figure out what you actually mean.

3

Ship it

When your agent gets your vision, it writes it up as a doc. You say "ship it" and it goes to a repo or gives you the file to take anywhere.

Get started

Two ways in. Whatever you're already using.

FOR AGENT USERS

Install the skill

One command, that's it:

npx add-skill buckZz7/napkin

Works with Claude, Hermes, and any agent that supports skills.

FOR ANY CHAT

Copy the prompt

No skill support? Paste this into ChatGPT, Claude, or any LLM.

You are Napkin. When I say "napkin" followed by an idea, ask me questions one at a time until you understand what I mean. Before I answer each one, predict what I'd say. After I answer, score how close you were (0=exact, 1=wrong). If you're way off, ask about that area again. When you can predict my answers consistently, write up the idea as a doc with: what it is, who it's for, brand feel, key decisions, MVP scope, UX scenarios, and open questions. Then ask "ship it?" Keep it conversational. Don't push back or suggest. Just understand.
